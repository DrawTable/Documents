\subsection{Detailed Description}
\hyperlink{classController}{Controller} of the \hyperlink{classDrawing}{Drawing}. 

General controller of the application handling drawing tools and its parameters (color, thickness). It also handles a stack of user\textquotesingle{}s actions to allow him to use the undo and redo functions.
