\subsection{Detailed Description}
The \hyperlink{classCameraManager}{Camera\+Manager} detects and lists user\textquotesingle{}s cameras. 

It tries to detect the ten first camera connected to the computer. It also provides a live view of every camera allowing the user to choose which camera he wants to use. When a user clicks on a window, a handler will be triggered and the associated camera is selected.

This class is a Singleton.
