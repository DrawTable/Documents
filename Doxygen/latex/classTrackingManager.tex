\subsection{Detailed Description}
Tracking Manager. 

This is the main manager for all the tracking part. It starts with the screen detection and once it\textquotesingle{}s done, it reads frames from the camera, find the position of the stylus using the \hyperlink{classLedDetector}{Led\+Detector} and mouve the mouse accordingly.
